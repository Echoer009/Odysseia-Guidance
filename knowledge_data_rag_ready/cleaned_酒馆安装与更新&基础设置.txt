### **SillyTavern (酒馆) 新手入门简介**

欢迎来到 SillyTavern 的世界！SillyTavern (酒馆) 是一个功能强大的 AI 角色扮演前端界面。

本教程将引导您完成从安装到基础设置的全过程，让您能够快速开始体验。

### **在 Windows 上安装 SillyTavern 的三种方法 (Git, 压缩包, Docker)**

对于 Windows 用户，有以下三种主流安装方式：

*   **Git 安装 (推荐) ⭐**
    *   **优点**: 功能最完整，支持一键更新。虽然初次设置稍显复杂，但一劳永逸。
    *   **教程**: 请参考类脑智识库的详细 Git 安装教程 👉 [点击查看](https://wiki.类脑.org/%E6%99%BA%E8%AF%86%E5%BA%93/sandbox/KKTsN/%E6%96%B0%E5%AE%9D%E5%AE%9D%E6%95%99%E7%A8%8B-flash)

*   **压缩包解压 (最简) 🥳**
    *   **优点**: 下载解压即可使用，非常适合快速体验。
    *   **缺点**: 无法一键更新，且可能因环境不完整导致无法安装扩展等问题。
    *   **操作步骤**:
        1.  从 **[此链接](https://codeload.github.com/SillyTavern/SillyTavern/zip/refs/tags/1.13.2)** 下载压缩包。
            *   注意：此链接固定下载 `1.13.2` 版本（截至2025年7月27日为最新稳定版）。
        2.  将其解压到一个 **不包含中文，空格或特殊字符** 的路径中（例如 `D:\SillyTavern`）。
        3.  进入解压后的文件夹，双击 `start.bat` 文件即可启动。

*   **Docker 安装 (折中) 🐳**
    *   **优点**: 安装难度和更新便利性介于 Git 和压缩包之间，同样支持一键更新。
    *   **教程**: 请参考类脑智识库的 Docker 部署教程 👉 [点击查看](https://wiki.类脑.org/%E6%99%BA%E8%AF%86%E5%BA%93/%E7%B1%BB%E8%84%91%E6%95%99%E7%A8%8B/%E6%90%AD%E5%BB%BA%E9%83%A8%E7%BD%B2/%E6%90%AD%E5%BB%BA%E9%83%A8%E7%BD%B2SillyTavern%E9%85%92%E9%A6%86%E4%B8%80%E9%94%AE%E9%83%A8%E7%BD%B2%E6%95%99%E7%A8%8B)

### **SillyTavern 首次启动后的五项必要基础设置**

安装完成后，首次启动需要进行一些关键设置，以确保最佳兼容性和体验。请跟随以下步骤操作：

1.  **初次进入 👤**
    *   首次启动会看到用户设置页面，直接点击右下角的 `保存` 即可，无需更改。

2.  **API 与模型设置 🔌**
    *   点击顶部工具栏第二个 **「插头」** 图标，进入连接设置。
    *   将 `API` 选项从 `AI Horde` 修改为 `聊天补全 (Chat Completion)`。
    *   将 `提示词后处理 (Prompt Post-processing)` 选项设置为 `严格 (Strict)`。

3.  **预设 (Preset) 设置 🎨**
    *   点击顶部工具栏最左边的 **「预设」** 图标，进入预设界面。
    *   **重要提示**: 酒馆自带的 `Default` 预设存在严重问题，**请不要直接使用它**。建议后续导入社区分享的优秀预设。
    *   **(可选操作)**: 点击预设界面右上角的绿色“锁链”图标，使其变为灰色。这可以解除预设与 API 设置的绑定，避免因忘记切换 API 源导致配置错乱。

4.  **世界信息 (Lorebook) 设置 📖**
    *   点击顶部工具栏第四个 **「书与地球」** 图标，进入全局世界信息设置。
    *   将 `上下文扫描深度百分比` 的滑块拉到 `100`。
    *   **取消勾选** `包括名称 `和 `匹配整个单词 `。

5.  **用户界面与体验设置 ⚙️**
    *   点击顶部工具栏第五个 **「人与齿轮」** 图标，进入用户体验设置。
    *   此处的选项较多。原教程建议勾选特定的推荐选项 (以红✔标记) 并取消勾选另一些 (以绿❎标记)。
    *   勾选: AI回复计时器，显示消息楼层，显示消息词符数，在机器人消息中允许{{user}}, 在机器人消息中允许{{char}}。
    *   取消勾选: click to edit, 自动修复Markdown, 禁止外部媒体, 在响应中显示标签
